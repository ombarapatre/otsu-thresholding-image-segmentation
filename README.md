# Otsu Thresholding for Image Segmentation

## Overview
This project implements Otsu Thresholding using Python and OpenCV for automatic image segmentation.

Otsu’s method automatically determines the optimal threshold value by minimizing intra-class variance and maximizing inter-class variance.

---

## Technologies Used
- Python
- OpenCV
- NumPy

---

## Methodology
1. Read grayscale image
2. Apply Otsu Thresholding
3. Segment foreground and background
4. Generate binary output image

---

## Formula Used

Within-class variance:

σ²w = Wbσ²b + Wfσ²f

Between-class variance:

σ²b = WbWf(μb − μf)²
